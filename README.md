# Beauty Academy Daniela Quesada

Landing Page desarrollada para Beauty Academy Daniela Quesada, enfocada en la promoción de servicios de belleza, micropigmentación, extensiones de pestañas y cursos profesionales.

## Descripción

Este proyecto consiste en una página web moderna, responsiva y accesible diseñada para mejorar la presencia digital de Beauty Academy Daniela Quesada. La plataforma permite a los usuarios conocer los servicios ofrecidos, explorar el catálogo de procedimientos, obtener información sobre cursos profesionales y contactar fácilmente con la academia.

## Características

- Diseño responsive para dispositivos móviles, tablets y escritorio.
- Sección Hero con llamada a la acción.
- Catálogo de procedimientos estéticos.
- Información sobre cursos profesionales.
- Galería de imágenes.
- Video promocional integrado.
- Ubicación mediante Google Maps.
- Enlaces directos a redes sociales.
- Navegación optimizada y accesible.
- Modo oscuro (Dark Mode).

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Google Maps Embed
- Diseño Responsive
- Accesibilidad Web (ARIA)
